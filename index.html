SUMMARY
dilerflow wanted to create a new & more efficient website to replace the old one
The previous website was based on the official dilerflow template resulting in low usability, poor conversion & lack of all necessary functions.
So, we decided to develop an appealing and functional website that would require less time to maintain, be more user-friendly, and increase user trust.
PROBLEM
Why to redesign old website?
  • Complicated CJM ( Customer Journey Map)
  • Low conversion rate
  • No personalized approach for customers
  • Bad website navigation
  • Lack of extremely important functionality for car dealer website (loan calculator, spare parts shop, etc)
  • Hard to compare different car models
PROJECT GOALS
/01
Create a Responsive Car Dealer Website
We aimed to create a well-thought-out responsive webdesign to deliver fast-loading experience on all devices.
/02
Impress with Efficient UX/UI Design
We strived to provide stunning UX/UI solutions within simple yet effective car dealer's website design.
/03
Deliver Conversion-Focused Design
We improved conversion rate of the website via call-to-action buttons, logical structure, advanced search functionality, comfortable menus & much more.
/04
Build a Functional Admin Panel
Our goal was to create a functional and user-friendly admin panel to allow managing content manually (for website administrators).
